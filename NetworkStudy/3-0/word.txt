■ ネットワーク層
IP
	相手先のアドレス情報をもとにしてパケットを送信する機能、通信するときの約束事（プロトコル）
IPヘッダ
	IPで扱う部分が書かれたデータ
ICMP
	Internet Control Message Protocolの略で通信が正常に行われているか確認する約束事
ICMPメッセージ
IPアドレス
	インターネット上でのコンピュータを特定する際に使うアドレス
		オクテット
			8ビットのこと
		ネットワーク部
			IPアドレスでのどこのネットワークであるかを示す
		ホスト部
			IPアドレスでのどこのコンピュータであるかを示す
		クラス
			IPアドレスの分類方法
		プライベートIPアドレス
			ローカルネットワーク内でしか通用しないIPアドレス
		グローバルIPアドレス
			全世界で通用するIPアドレス
ネットワークアドレス
	どこのネットワークであるかを示すネットワーク部のみのIPアドレス
ブロードキャストアドレス
	そのアドレスに送ると、ネットワークにつながっている全てのの機器に届くようになっているアドレス
ループバックアドレス
	自分自身を示すIPアドレス
サブネット
	1つの大きなネットワークを分割して作った小さいネットワーク
サブネットマスク
	IPアドレスのどの部分がネットワーク部で、どの部分がホスト部かを示す情報
		プレフィックス表記
			IPアドレスとサブネットマスクを同時に表現する書き方
デフォルトゲートウェイ
	通信において宛先がわからないときに取りあえず送る送り先
IPv6
	IPアドレスのことIPv4より割り当てれる数が多い
		サブネットプレフィックス
			基本的に前半64ビットがサブネットプレフィックスにあたるIPv4のネットワーク部にあたる
		インターフェイスID
			リンク上でホストを識別するため
 

ルータ
	コンピュータネットワークにおいて、データを2つ以上の異なるネットワーク間に中継する通信機器ネットワーク層において、どのルートを通してデータを転送すべきかを判断するルート選択機能を持つ
		コンソール接続
			ルータのコンソールポートとPCをコンソールケーブルで接続する方法
		VTY接続
			物理的に離れたところからネットワークを通じて仮想的に接続する方法
		AUX接続
			ルータのAUXポートとモデムを接続し、電話回線経由で接続する方法
		ルーティングテーブル
			ルーティングの情報が記載されている
				ネクストホップ
					ルーティングにおける、データの転送先
		ARPテーブル
				ARP
					IPアドレスからMACアドレスを調べる仕組み
		コアルータ
			基幹ネットワークにおいてメインで用いられるルータ、通信キャリアやISPネットワークの通信を中継するために利用されるルータ
		センタールータ
			企業で中心的に利用されるルータ、拠点となる本社や、データセンター等の企業内で中心となる場所を接続するためのネットワークを構成する
		エッジルータ
			企業で利用されるルータ、中心となる拠点を除く、営業所や支社等の拠点間を接続するネットワークを構成する
		ブロードバンドルータ
			家庭環境やSOHO環境等、インターネット回線に接続することを目的として利用されるルータ



ルーティング
	　異なるネットワークにパケットを送信する際に最適な経路を決めること
		直接接続
			ルータが直接接続しているネットワークのルート
		スタティックルーティング
			管理者が宛先ネットワークへの最適なルートを手動で設定したルート
		ダイナミックルーティング
			ルーティングプロトコルにより動的に追加されるルート
				ディスタンスベクタ型
					ホップ数（=宛先に到達するまでに経由するデバイス数）をメトリックとし、経路を選ぶ方式
				リンクステート型
					各ルータが全体の接続情報を見て、最適経路を選択する方式
				ハイブリッド型
					ディスタンスベクタ型をベースに、リンクステート型を組み合わせた方式（Ciscoが独自に拡張した方式）
 

■ トランスポート層

コネクション型通信
		ACK
			確認応答番号が有効であることを示すフラグの一つ
コネクションレス型通信
		TCP
			(Transmission Control Protocol)データを送信する前に通信相手と論理的な通信路を確立する。
		スリーウェイハンドシェイク
			TCPによる接続（コネクション）を確立するための手順。パケットの送受信を3回行うことからスリーウェイハンドシェイクと呼ぶ
		Telnet
			ネットワーク上の機器を遠隔操作するためのプロトコル
		HTTP
			Webブラウザなどでホームページを閲覧する際にWebページを送受信するためのプロトコル
		FTP
			ネットワーク経由でファイルを転送するためのプロトコル
		SMTP
			電子メールを受信するために使用するプロトコル
TCPヘッダ
	信頼性を提供するための様々なフィールドが設定されている
UDP
		RTP
			インターネット上で音声や動画をリアルタイムに転送するプロトコル
		SNMP
			ネットワーク機器やサーバをネットワーク経由で監視し管理するためのプロトコル
		DHCP
			ユーザのコンピュータにIPアドレスなどの情報を自動的に割り当てるプロトコル
		TFTP
			UDPを用いて簡易的にファイルを転送するためのプロトコル
		RIP
			UDPのブロードキャスト・データ・パケットを用いて、経路情報を隣接ルータにアナウンスするプロトコル
UDPヘッダ
	TCPに比べて即時性が必要なもので信頼性は提供されていない
